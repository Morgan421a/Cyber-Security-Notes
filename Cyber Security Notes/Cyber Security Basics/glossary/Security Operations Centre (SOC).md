A team of Cyber Sec professionals who monitor the network and its systems to detect malicious cyber sec events

**Some areas of focus for them are:**

- [[Vulnerabilities]]
- [[Security Policy]] Violations e.g. a user uploading company data to an online storage device
- Unauthorised Activity
- Network Intrusions

==**Threat intelligence:**==

A task within SOC.

Their task is to gather information about actual and potential enemies.

Threat intelligence collects info to help the company prepare against potential enemies. 

Data is collected, processed and analysed from local sources such as network logs and public sources like forums.

Data processing sorts the collected data into a form more suitable for analysis. 

The analysed data is then used to find more info about the attackers and their motives while also creating a list of recommendations and actionable steps.

This process allows for response strategies to be formed as well as giving us the ability to predict an attacker's activity.
