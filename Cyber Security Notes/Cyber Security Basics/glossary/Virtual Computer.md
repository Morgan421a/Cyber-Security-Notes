1. Virtual Computer - A virtual Machine that is running on someone else's hardware and is accessed over the internet, uses the exact same technology as a VM. 
	   - In **cloud computing** - Clients can utilise for their apps or services in order to provide users with access and interaction with said app or service typically via a GUI

2. Virtual Server - Server created using a VM that allows clients to setup a server for their app, service or software to which users can send requests and receive responses
