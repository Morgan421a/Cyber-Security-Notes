Dynamic Host Configuration Protocol

- Process used to auto assign IP addies to devices on a network via requests and replies between device and DHCP server
1. DHCP Discover - device requests IP addy if a DHCP server is on the network
2. DHCP Offer - Server replies with available IP addy
3. DHCP Request - Device requests offered IP addy
4. DHCP ACK - Server acknowledges process complete and device can use IP addy
