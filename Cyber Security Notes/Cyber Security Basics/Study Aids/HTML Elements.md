`<!DOCTYPE HTML>` - Defines page as an HTML5 document, helps with standardisation across browsers and tells the browser to use HTML5 to interpret the page

`<html>` - the root element of the HTML page, all other elements come after this one

`<head>` - Contains information about the page i.e. page title

`<body>` - Defines the document's body, only the content in the body is shown in the browser

`<h1>` - Defines a large heading

`<p>` - Defines a paragraph

`<button>` - Defines a button

`<ul>` - Defines and unordered list - using bullet points for each entry

`<ol>` - Defines and ordered list - using sequential numbers or letters for each entry

`<li>` - Defines an item with either a `<ul>` or `<ol>` list and must be used for each entry
