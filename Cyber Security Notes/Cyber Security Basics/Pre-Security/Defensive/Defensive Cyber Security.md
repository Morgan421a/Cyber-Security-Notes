Two main tasks:

- Preventing intrusions from occurring

- Detecting Intrusions when they occur and responding properly

Blue teams are part of the defensive security landscape

Tasks relating to defensive security include:

- Training users regarding cyber security to help protect from attacks targeting their systems

- Documenting and managing assets to ensure we're aware of what systems and devices must be managed and protected

- Making sure that computers, servers and network devices are updated and patched properly to protect against known weaknesses

- Setting up security devices such as Firewalls and IPS

- Setting up logging and monitoring devices to detect malicious activities and intrusions.

Two main topics in defensive security include:

- SOC

- DFIR

A SIEM tool gathers security related information and events from various sources and presents them in one dashboard. If the SIEM finds something suspicious an alert will be generated.
