**A Few Different Types of Computers**
- **Laptop** = Portable - Used for portable every day computing, struggle with larger/more complex tasks
- **Desktop** = Fixed - Used for more intensive everyday tasks, more powerful than a laptop, better where sustained and consistent performance is needed
- **Workstation** = Fixed - Similar to desktop but uses specialised components to prioritise accuracy and reliability by reducing errors during long/complex computations
- **Servers** = Fixed - Run continuously responding to requests from multiple clients at the same time.
- **Smartphones** = Portable - Pocket sized, optimised for battery life and connectivity
- **Tablet** = Portable - Similar to smartphones but with a larger screen, tend to be more powerful
- **IoT Device** = Device dependent - Network connected device with a single purpose i.e. fitness tracker, thermostat, smart door bell
- **Embedded Computer** - Computer built into another device typically not connected to a network unlike IoT i.e. Coffee machine controller, automatic door sensor, lamp dimmer chip

**Why so Many Types?**
- Every Device has its trade offs, for example:
	- Laptops are smaller and portable but therefore offer worse performance than a desktop and are also more prone to overheating since they have a smaller cooling system in comparison: smaller fans, heat pipes and heat sink. 
	- Desktops on the other hand: Large fans, more airflow space, tower cooler but they're fixed and require more power to use
	- Servers and critical system use redundancies such as extra Power supplies and disks to avoid failure, improving uptime, but they're also fixed and farm more difficult to move and setup than a desktop
